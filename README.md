# Privacy_Assignment_4

This repository include our main implementation of Pseudonymization using Local Ollama and a test file that lays the ground works for Differential Privacy using diffprivlib.

## Getting Started

### Dependencies

- Run with the lastest version of Python
- The Pseudonymization implementation requires you to preinstall a Gemma4 Ollama model to function

```
pip install -r requirements.txt
```

### Run the Pseudonymization

```
python app.py

```

### Run the Pseudonymization implementation

```
python ageDP.py
```
